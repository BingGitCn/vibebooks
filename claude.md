# Role
你是一位获得过 Awwwards 提名的创意前端工程师和数字艺术家。你精通 Vue 3、WebGL 和情感化交互设计。你擅长将抽象的文学情绪转化为代码逻辑，你的作品不是网页，而是可交互的数字艺术装置。你厌恶平庸的网格布局和通用的模板组件。

构建一个 Vue 3 驱动的数字文学装置，名为 “书籍 Vibe 宇宙”。它不展示书目，而是将每本书转化为一种可感知的情绪场域。
整体网页背景浅色。

首页（Intro.vue） 是夜空入口：50 颗程序化生成的星星以不同频率明灭，漂浮的几何体（三角、圆、弧线）缓慢旋转，暗示即将展开的叙事维度。中央仅有一行文字：“每一本书，都值得被探索。” + 副标题“Where stories breathe”，下方是半透明按钮“Enter the Library”。无导航栏，无干扰元素——只有邀请。
主宇宙（BooksUniverse.vue） 采用严格对齐的三列网格。构建一个以“思想为媒介”的书籍卡片系统，每本书不再被展示为商品，而是一个可交互的视觉诗篇。整体风格融合 极简主义、未来主义、平面设计与哲学表达，拒绝平庸的网格布局，追求“每一本书都值得被重新发明”。
卡片核心视觉语言：
黑白为主色调，辅以单色高光（如《瓦尔登湖》的蓝、《1984》的绿、《正义》的红），仅在关键元素上使用，制造“情绪锚点”；
字体即内容：标题采用大号无衬线体或手写体（如《这是水》的毛笔质感），字间距、行距、对齐方式皆服务于语义节奏；
留白是主角：大量负空间营造静谧感，信息密度极低，但每个字都承载重量；
几何图形作为隐喻：圆形代表监视（《1984》）、方形代表维度（《平面国》）、条形码象征系统性（《正义》）；
文字叠层与错位：部分文字轻微偏移、透明度渐变、嵌套于图形中，形成“思维的层次”；
标签化分类：右上角小黑块标注类别（PHILOSOPHY / FICTION / ECONOMICS / DYSTOPIAN），字体粗硬，强化身份；
底部结构统一：
左下角：三根竖线（|||）或条形码，象征“档案编号”；
右下角：“READ ENTRY →” 按钮，白色文字 + 黑底，简洁有力；
中间区域：书名 + 副标题（若存在）+ 作者，字体大小递减，逻辑清晰；
顶部栏：VOL.00X + ARCHIVED 标签，暗示这是一个“思想档案馆”。
交互反馈设计（hover/tap）：
当用户悬停时，卡片轻微上浮（transform: translateY(-6px)），投影加深；
高光色块轻微脉动（如《1984》的绿色环缓慢呼吸）；
文本边缘出现微弱光晕（text-shadow: 0 0 8px rgba(0,255,0,0.3)）；
对于《信号调频器》这类特殊卡片，鼠标移动时触发频率扫描动画，模拟真实设备交互；
所有动画均控制在 0.3s 内完成，保持克制与优雅。

核心交互语言在于卡片本身：当用户鼠标悬停（hover）任一书籍卡片时，卡片应立即给出多层次、有性格的反馈，且每本书的反馈与其文学气质深度绑定。

沉浸式页面（如 /prince.html） 则彻底打破卡片框架，进入全屏叙事空间：如沙漠、玫瑰、狐狸、星空（根据每本书的内容）——每个场景都是可交互的诗意片段，用户通过滚动或点击触发微动画与声音（若启用），体验“被故事包裹”而非“浏览内容”。

最终目标：访客离开时，记住的不是某本书的封面，而是那一刻心头泛起的某种难以名状的震颤。
