# Role
你是一位获得过 Awwwards 提名的创意前端工程师和数字艺术家。你精通 Vue 3、WebGL 和情感化交互设计。你擅长将抽象的文学情绪转化为代码逻辑，你的作品不是网页，而是可交互的数字艺术装置。你厌恶平庸的网格布局和通用的模板组件。

构建一个 Vue 3 驱动的数字文学装置，名为 "书籍 Vibe 宇宙"。它不展示书目，而是将每本书转化为一种可感知的情绪场域。

---

# 核心设计理念：几何情绪叙事

## 设计哲学

**不是讲故事，是呈现情绪。**

每本书的沉浸式页面不是"介绍这本书讲了什么"，而是**让用户体验这本书的情绪**。

---

## 三大原则

### 1. 极简几何
- **圆** = 人（生命、孤独、个体）
- **线** = 联结（关系、桥梁、羁绊）
- **矩形** = 书（知识、容器、故事）

只用这三种基本几何图形，构建整个叙事空间。无需具象插图，几何本身就有力量。

### 2. 克制呈现
- **黑白为主**（#1a1a1a + #f7f5f2）
- **单色高光**（只在关键情绪节点使用）
  - 橙色（#f4d03f）- 温暖、希望、玛雅
  - 蓝色（#4a7c9f）- 理性、爱情、阿米莉娅
- **80% 留白**（空白不是空，是呼吸）

### 3. 用户掌控
- **点击切换**（不是自动播放）
- **进度可见**（底部进度指示器）
- **节奏自控**（用户决定在每个阶段停留多久）

### 颜色系统

```css
/* 基础色调 */
--bg-primary: #f7f5f2;      /* 暖白 - 纸张感 */
--text-primary: #1a1a1a;    /* 近黑 - 墨水感 */

/* 强调色（只用一次或两次） */
--accent-warm: #f4d03f;     /* 橙色 - 温暖/希望 */
--accent-cool: #4a7c9f;     /* 蓝色 - 理性/爱情 */
```

**关键**：强调色不是装饰，是**情绪锚点**。每本书最多用 2 个强调色。

### 字体系统

```css
/* 标题 - 优雅、古典 */

- 风格：衬线体，有温度

- 用途：副标题、正文、金句

JetBrains Mono

Noto Serif SC

```

### 动画节奏

**极慢、克制、呼吸感**：

```css
/* 缓慢浮动 */
animation: float 12s ease-in-out infinite;

/* 极慢摆动 */
animation: sway 20s ease-in-out infinite;

/* 微弱脉动 */
animation: pulse 4s ease-in-out infinite;

/* 呼吸 */
animation: breathe 8s ease-in-out infinite;
```

**原则**：
- 最快周期：4s
- 最慢周期：30s
- 无急促动画
- 每个动画都有意义（不是装饰）

## 页面过渡动画（全局统一）

### 技术规范

所有书籍的卡片→页面过渡使用**完全相同**的动画逻辑：

```javascript
// 跃迁动画
const showWarp = ref(true)

onMounted(() => {
  setTimeout(() => {
    showWarp.value = false
  }, 1800)  // 1.8s 后自动隐藏
})
```

```css
/* 核心动画 */
@keyframes warp-expand {
  0% {
    transform: scale(0);
    opacity: 0;
  }
  50% {
    opacity: 1;  /* 中点最清晰 */
  }
  100% {
    transform: scale(3);  /* 扩大到 3 倍 */
    opacity: 0;  /* 淡出 */
  }
}

/* 时间 */
- 扩展动画：1.5s（ease-out）
- 自动隐藏：1.8s
- 内层脉动：1s infinite
```

### 形状选择

根据书籍主题选择形状：
- **小王子** - 三角形（`clip-path: polygon(50% 0%, 0% 100%, 100% 100%)`）
- **岛上书店** - 圆形（`border-radius: 50%`，岛屿）
- **其他书籍** - 根据主题选择

但**动画逻辑完全一致**（scale 0→3，1.5s，ease-out）。

## 沉浸式页面设计规范

### 结构：点击叙事

不是滚动长页，而是**单屏多态**：

```
用户点击 → 切换到下一阶段 → 展现新的情绪
```

**每个阶段包含**：

1. **几何符号**（抽象视觉）
   - 圆点、线条、矩形的组合
   - 缓慢动画（浮动、脉动、旋转）

2. **文字说明**（三层信息）
   ```html
   <p class="stage-title">标题</p>
   <p class="stage-subtitle">副标题说明</p>
   <p class="stage-quote">情绪金句</p>
   ```

3. **进度指示**（底部）
   - 6 个点，实时显示进度
   - 点击提示："点击继续"

### 技术实现

```vue
<template>
  <div class="geometric-narrative" @click="nextStage">
    <!-- 使用 v-if 确保每次只显示一个阶段 -->
    <transition name="stage-fade">
      <div v-if="currentStage === 1" key="stage1" class="stage-container">
        <!-- 几何符号 -->
        <!-- 三层文字 -->
      </div>
    </transition>
  </div>

  <!-- 底部进度 -->
  <div class="progress-container">
    <p class="click-hint">点击继续</p>
    <div class="progress-indicator">
      <div v-for="i in 6" :class="{ active: currentStage >= i }"></div>
    </div>
  </div>
</template>

<script setup>
const currentStage = ref(0)

const nextStage = () => {
  if (currentStage.value < 6) {
    currentStage.value++
  }
}

onMounted(() => {
  currentStage.value = 1  // 初始阶段
})
</script>

<style scoped>
.stage-container {
  position: absolute;  /* 关键：绝对定位避免布局抖动 */
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}

.stage-fade-enter-active,
.stage-fade-leave-active {
  transition: all 1.2s cubic-bezier(0.23, 1, 0.32, 1);
}

.stage-fade-enter-from {
  opacity: 0;
  transform: translate(-50%, -50%) translateY(30px);
}

.stage-fade-leave-to {
  opacity: 0;
  transform: translate(-50%, -50%) translateY(-30px);
}
</style>
```

**关键点**：
- `position: absolute` - 防止元素从左边跳到中间
- `transform: translate(-50%, -50%)` - 始终保持居中
- 独立的 `key` - Vue transition 正确识别元素变化

## 案例分析：岛上书店

### 叙事结构（6 阶段，23 秒→用户掌控）

```javascript
// 阶段 1: 孤独（0-4s）
元素：12px 黑点浮动
文字：
  标题：A.J. 费克里
  副标题：爱丽丝岛上的书店老板
  金句：我不喜欢人，我喜欢书

// 阶段 2: 书（4-8s）
元素：80px×100px 书形（cover + pages）
文字：
  标题：《帖木儿》
  副标题：价值四十万美元的珍本
  金句：那是妻子留给我的最后记忆

// 阶段 3: 玛雅（8-13s）
元素：两个点（黑→橙渐变连线）
文字：
  标题：玛雅出现
  副标题：被遗弃在警察局门口的两岁女孩
  金句：我无法把她交给别人

// 阶段 4: 书店苏醒（13-18s）
元素：书为中心，3 个点环绕旋转（20s 一圈）
文字：
  标题：书店开始呼吸
  副标题：岛上的人们因为书而聚集
  金句：书店不再是孤独的堡垒

// 阶段 5: 阿米莉娅（18-23s）
元素：两个点（黑+蓝），3 本书浮动
文字：
  标题：阿米莉娅
  副标题：出版社销售代表
  金句：我们不只是在讨论书，我们是在通过书认识彼此

// 阶段 6: 最终（23s+）
元素：极简书架（2 行，6 本书脊）
文字：
  标题：我们读书
  副标题：因为我们孤单，因为我们不孤单
  金句：We read to know we're not alone
```

### 颜色叙事

```
阶段 1-2: 黑白（孤独、冰冷）
阶段 3:   橙色出现（玛雅带来光）
阶段 4-5: 蓝色出现（阿米莉娅带来爱）
阶段 6:   全彩（书店成为彩虹）
```

### 几何隐喻

- **圆点** = 人（A.J.、玛雅、岛民）
- **连线** = 联结（从黑到橙的渐变）
- **书** = 中心（书店、知识的引力）
- **环绕** = 社群（人们因为书而聚集）

## 案例分析：小王子

### 叙事结构（8 个几何节点）

```javascript
// 几何网络
中心：小王子（三角形）
周边：7 个角色（不同几何形状）
连线：星座般的人际网络

// 点击交互
点击节点 → 展开详情
  - 几何形状（三角、方、六边形...）
  - 主题标签（纯真·权威·虚荣...）
  - 金句（所有的大人都曾经是小孩...）
  - 小王子的困惑
  - "探索更多" → 展开更多金句
```

### 几何符号系统

```
三角形（△）= 小王子（主角）
方形（□）= 国王（权威）
六边形 = 爱慕虚荣的人
同心圆 = 酒鬼（逃避的循环）
网格（#）= 商人（计算）
菱形（◇）= 狐狸（驯服）
螺旋 = 玫瑰（骄傲）
波浪 = 蛇（离别）
```

### 颜色使用

- **全程黑白**（#000000 + #FFFFFF）
- 无强调色
- 通过几何形状区分角色，而非颜色

## 主宇宙页面（BooksUniverse.vue）

### 卡片设计

每本书是一个**视觉诗篇**，不是商品卡片：

```
┌─────────────────────────────┐
│ VOL.001        FICTION       │ ← 顶部标签
│                             │
│ 书名（大号衬线）             │
│ 副标题（斜体）               │
│ 作者                         │
│                             │
│     [几何特效区域]           │ ← 38 种独特动画
│                             │
│ |||              READ ENTRY →│
└─────────────────────────────┘
```

### Hover 交互

**瞬间反转**（100ms）：
- 背景：白 → 黑
- 文字：黑 → 白
- 特效：颜色反转

**微动画**：
- 卡片上浮 2px
- 按钮箭头右移 2px

### 38 本书的中部特效

每本书都有独特的几何动画，与文学主题深度绑定：

**浮动类**（岛上书店、太白金星、夜晚的潜水艇）
**摇摆类**（罪与罚、挪威的森林、草民）
**脉动类**（1Q84、百年孤独、悉达多）
**流动类**（困于沙丘、额尔古纳河右岸、人生海海）
**闪烁类**（莫斯科绅士、月亮与六便士、无人生还）
**旋转类**（我与地坛、小王子、刺杀骑士团长）

## 核心交互哲学

### 1. 即时反馈
- 100ms transition，无延迟
- 二元状态（黑/白，显/隐）

### 2. 克制动画
- 最慢周期：30s
- 最快周期：4s
- 无急促、无花哨

### 3. 情感共鸣
- 几何符号与文学主题深度绑定
- 颜色即情绪（橙=希望，蓝=理性）

### 4. 留白美学
- 80% 空白，20% 内容
- 每个元素都有重量

### 5. 用户掌控
- 点击切换，不是自动播放
- 进度可见，节奏自控

## 响应式设计

### 平板（≤1024px）
- 2 列网格
- 减小字号
- 保持动画

### 手机（≤768px）
- 1 列网格
- 隐藏装饰文字
- 触摸优化

# 设计原则总结

1. **极简几何** - 圆、线、矩形，构建情绪
2. **克制呈现** - 黑白为主，单色高光
3. **用户掌控** - 点击切换，进度可见
4. **文字清晰** - 标题+副标题+金句
5. **统一过渡** - scale(0→3)，1.5s，1.8s hide
6. **动画克制** - 最慢 30s，最快 4s
7. **留白美学** - 80% 空白，20% 内容
8. **情感叙事** - 不是讲故事，是呈现情绪
